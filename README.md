# Batman Text Mining Project

This project explores text mining and analysis techniques using
a Batman Trilogy dataset. The goal was to extract insights from
textual data using R and present findings in a reproducible format.

## Project Files
- `TM_project.Rmd` – R Markdown source containing code, analysis, and commentary
- `TM_project.html` – Rendered HTML report with results and visualizations

## Tools & Techniques
- R
- R Markdown
- Text mining / text analytics
- Data preprocessing and analysis
- Machine Learning

## Skills Demonstrated
- Text analytics workflow
- Reproducible research
- Analytical reasoning
- Data storytelling
